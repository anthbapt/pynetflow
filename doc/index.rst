=============================================================================================
Pynetflow - Python Ricci Flow package
=============================================================================================

.. image:: https://img.shields.io/pypi/v/pynetflow.svg
    :target: https://pypi.python.org/pypi/pynetflow

.. image:: https://img.shields.io/pypi/pyversions/pynetflow.svg

.. image:: https://readthedocs.org/projects/pynetflow-doc/badge/?version=latest
    :target: https://pynetflow-doc.readthedocs.io/en/latest/?badge=latest
    
.. image:: https://github.com/anthbapt/pynetflow/workflows/CI/badge.svg
    :target: https://github.com/anthbapt/pynetflow/actions?query=branch%3Amaster+workflow%3ACI
    

Pynetflow package aim is to compute the Ricci curvature of a given network and the Ricci flow process associated to a metric defined on the network. 
Then several methods derived from Ricci flow can be used, such as communition detection, and network embedding.
